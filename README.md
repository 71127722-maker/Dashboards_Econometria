# Dashboard Banco Mundial

Dashboard interactivo desarrollado en R Shiny usando datos del Banco Mundial.

## Características

- KPIs económicos
- Series temporales
- Comparación entre países
- Mapas interactivos
- Tablas dinámicas

## Tecnologías usadas

- R
- Shiny
- Plotly
- Leaflet
- WDI
- DT
